# Search over 1,000,000 recipes through 'Forkify'. Explore, upload, and add recipes to your collection.

---

## Installation

Install forkify with npm

```bash
  npm install
  npm run start
```

---

## Deployment

To deploy this project run

```bash
  npm run build
```

---
